# 掛け算トレーニング
## About
このソフトウェアは簡易的な掛け算練習を目的に作成されています  
問題の答えを>>>に続いて入力しEnterキーを押してください  
正解であれば☑️が、間違っていたら❎が表示されます  
練習を終える際は直接ウィンドウを閉じてください  
CommandPrompt等をご利用の方は"""chcp 65001"""コマンドを実行してからご利用ください
WindowsTerminalをご利用していただくと、より快適にご利用いただけます。
